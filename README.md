## Getting Started

1. Clone project and install dependencies

```bash
$ git clone https://github.com/eric-bach/pecuniary.git
$ cd pecuniary
$ npm install
```

2. Start the project

```bash
$ npm start
```

3. Run the app

```
http://localhost:3000/
```
